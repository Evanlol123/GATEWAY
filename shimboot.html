Shimboot

Steps:

  1. Download the file here
  2. Flash it to a usb. Learn how to do that here
  3. Dev mode execute the usb on your chromebook. Learn how to do that here
  4. Select debian
  5. Username/Password is user/user

How does this work?

While in dev mode you can boot rma shims. This uses an rma shim then edits it to be linux and everything it run from your usb drive and has no affect on downloads/space on your chromebook.
