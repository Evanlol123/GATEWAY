Steps:

  1. Download the raw file here
  2. Search through your apps and find "Sparkvue"
  3. Click the app
  4. Let it load, then click "ok"
  5. Click "Open Saved Experiment"
  6. Select the file you just downloaded

How does this work?

While in dev mode you can boot rma shims. This uses an rma shim then edits it to be linux and everything it run from your usb drive and has no affect on downloads/space on your chromebook.
